# abhay-skills

A collection of Claude plugins and skills by **Abhay Singh Nagarkoti** — covering automation, content creation, sales intelligence, and more.

## 🔌 Load into Claude Code

Add this marketplace to Claude Code by running:

```bash
claude plugin add https://github.com/abhaysingh1122/abhay-skills
```

Or paste the marketplace URL directly in Claude Code's plugin settings:

```
https://github.com/abhaysingh1122/abhay-skills
```

---

## 📦 Plugins

### 🔧 n8n
Build, test, and deploy n8n workflows via REST API with incremental testing.

### 📋 n8n PRD Generator
Convert discovery call transcripts into concise n8n Automation Blueprints with an interactive question flow.

### 📧 Email Sequence
Expert email marketing and automation skill for creating email sequences, drip campaigns, nurture sequences, onboarding flows, and lifecycle email programs.

### 🎨 Excalidraw
Create visual presentations, slide decks, and explanatory diagrams in Excalidraw. Supports file output or direct Chrome browser injection.

### 📊 Infographic
Generate professional LinkedIn-style infographics using Gemini AI image generation. Creates clean, memorable visuals with flat colors and hand-drawn touches. Supports brand guidelines, series creation, and iterative refinement.

### 🔍 SEO Audit
Run comprehensive SEO audits on websites using seomator CLI — 16 categories, 148 rules covering technical SEO, performance, accessibility, security, and more.

### 🎬 Video
Video editing with FFmpeg and Remotion — stitching, transitions, captions, teasers, and more.

### 💼 Sales
Research and score B2B leads, craft personalized outreach sequences, and build client success stories. Turn discovery call transcripts into actionable automation blueprints.
